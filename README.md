This repository contains clean, practical implementations of design patterns in Java, created to strengthen Low Level Design (LLD) understanding and interview readiness.

The focus is on:

Clear intent of each pattern

Realistic use cases

Clean object-oriented design

SOLID principles

Thread-safety where applicable
